# Titanic Survival Analysis

This project explores the Titanic dataset using Python, Pandas, Seaborn, and Scikit-learn.  
It includes exploratory data analysis (EDA), visualizations, and a logistic regression model.

## Project Structure
- `data/` → dataset (`tested.csv`)
- `notebooks/` → Jupyter notebook for exploration
- `src/` → Python script for reproducible runs
- `results/` → saved plots and profiling report
- `requirements.txt` → dependencies
- `.gitignore` → excludes venv and cache files

## How to Run
1. Clone the repo
2. Create a virtual environment
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
